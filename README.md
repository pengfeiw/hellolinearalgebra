# 你好，线性代数

关于线性代数的几何意义的教程。

在线地址：https://pengfeiw.github.io/hellolinearalgebra/

## 本地运行

1. clone 代码

```bash
git clone https://github.com/pengfeiw/hellolinearalgebra.git
```

2. 安装依赖

```bash
npm install
```

3. 启动本地服务

```bash
npm run dev
```

4. 打开 [http://localhost:3100](http://localhost:3100)
